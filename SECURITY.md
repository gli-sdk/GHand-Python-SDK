# Security Policy

## Supported Versions

The following versions of GHand SDK are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in GHand SDK, please report it responsibly.

**Do not** open a public issue for security vulnerabilities.

Instead, please send an email to [support@glitech.com](mailto:support@glitech.com) with the following information:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.
- Any suggested fixes or mitigations.

We will acknowledge receipt of your report within 48 hours and will work to provide a fix as soon as possible.

## Disclosure Policy

We follow a coordinated disclosure policy:

1. We will confirm the vulnerability and begin working on a fix.
2. We will notify affected users if necessary.
3. We will release a patched version.
4. We will publicly disclose the vulnerability with credit to the reporter (unless anonymity is requested).
