安装与配置指南
================

系统要求
--------
- Python 3.10 to 3.13
- Windows 10/Linux Ubuntu 18.04+/macOS 10.15+
- 推荐内存：8GB+
- 存储空间：1GB+

安装SDK
-------
.. code-block:: bash

   # 使用pip安装
   pip install ghand
   
   # 或者从源代码安装
   git clone https://gitee.com/glitech/ghand-sdk.git
   cd ghand-sdk
   python setup.py install

---------------------


卸载SDK
---------------------
.. code-block:: bash

   pip uninstall ghand
