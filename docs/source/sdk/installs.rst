Installation and Configuration Guide
====================================

System Requirements
-------------------
- Python 3.10 to 3.13
- Windows 10 / Linux Ubuntu 18.04+ / macOS 10.15+
- Recommended memory: 8GB+
- Storage space: 1GB+

Install SDK
-----------
.. code-block:: bash

   # Install using pip
   pip install ghand

   # Or install from source
   git clone https://github.com/gli-sdk/GHand-Python-SDK
   cd GHand-Python-SDK
   python setup.py install
