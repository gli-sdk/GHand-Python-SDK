# 自适应抓取功能需求文档

**日期**: 2026-04-17  
**功能**: 基于触觉反馈的自适应抓取  


## 1. 背景
具身机器人本体在使用灵巧手时，会用灵巧手取抓取物品，在搬运或移动的过程中，物品有滑脱的风险，而通过触觉传感器，能感知物体的滑移趋势，通过反馈微调关节角度，增加法向力，减少滑脱的情况。

## 2. 总体功能要求：
灵巧手运控算法稳定抓取模块需具备“感知-判断-控制-反馈”闭环能力，集成力感知、滑移检测、力调节三大核心功能，无需人工干预，自动适配不同材质、不同形状的抓取物体（预设常见物体参数库，支持参数扩展），确保抓取过程稳定、安全、可控。

## 3. 详细功能需求：
### 3.1 稳定抓持功能
#### 3.1.1 抓取力自适应调节
- 功能描述：算法根据抓取物体的材质、重量、形状（预设参数或实时识别），自动计算并输出最优初始夹持力，避免夹持力过小导致物体滑移，或夹持力过大损伤物体。
- 具体要求：  
  1. 预设物体参数库：包含常见物体（如金属块、塑料件、玻璃制品、柔性织物）的材质硬度、重量、安全夹持力范围，支持手动添加、编辑物体参数。
  2. 初始夹持力计算：根据物体重量、材质硬度，按公式（初始夹持力=物体重量×安全系数+基础夹持力，安全系数可配置，范围1.2-2.0）自动计算初始夹持力或预设夹持力，确保初始夹持力处于物体安全夹持范围内。
  3. 夹持力校准：抓取过程中，实时采集手指力传感器数据，与预设安全范围对比，若初始夹持力偏差超过±5N，自动微调夹持力至合理范围。
- 判定标准：初始夹持力落在物体预设安全范围内，抓取后10s内物体无滑移、无明显变形（易损物体无破损）。

#### 3.1.2 抓取姿态稳定控制
- 功能描述：抓取过程中，算法实时调节灵巧手各关节角度，确保物体姿态保持预设角度（如水平、垂直），避免物体倾斜、翻转，实现姿态稳定。
- 具体要求：
  1. 人眼观测：稳定抓取后，人眼观察抓取物品的倾斜角度、翻转角度。
  2. 视觉传感器（可选）：实时采集物体姿态数据（倾斜角度、翻转角度）
- 判定标准：抓取后，人眼观察无明显倾斜、翻转或视觉传感器测量物体姿态倾斜角度≤5°、反转角度≤5°，持续保持稳定，无明显晃动。

#### 3.1.3 损伤防护控制
- 功能描述：针对易损物体（如玻璃、豆腐、陶瓷、柔性件），算法具备损伤防护机制，避免夹持力过大导致物体破损、变形。
- 具体要求：
  1. 易损物体识别：支持通过参数库标记易损物体，或通过视觉传感器（可选）识别易损物体特征，自动切换至“损伤防护模式”。
  2. 夹持力限制：易损物体的最大夹持力不得超过预设安全阈值（可配置，根据物体材质设定，如玻璃制品≤15N），一旦接近阈值（达到阈值的90%），自动减缓夹持速度，直至达到阈值后停止加压。
  3. 关节动作限制：损伤防护模式下，灵巧手关节动作速度降低30%，避免动作过快产生冲击，损伤物体。
- 判定标准：易损物体抓取后无破损、无变形，夹持力始终≤预设安全阈值。

## 3.2 动态增稳功能
### 3.2.1 滑移趋势检测
- 功能描述：抓取过程中（包括静态夹持、动态移动过程），实时检测物体是否存在滑移趋势，及时触发增稳策略，避免物体滑移掉落。
- 具体要求：
  1. 滑移检测参数：通过手指力传感器采集摩擦力数据，综合判断滑移趋势（滑移趋势判定：摩擦力≤物体重力×摩擦系数-安全余量，或手指与物体相对位移≥0.3mm）。
  2. 检测频率：≥60Hz，确保滑移趋势及时被捕捉，避免延迟导致物体滑移。
  3. 误判防护：排除因手指微小振动、传感器噪声导致的误检测，设置防抖阈值（如连续3个采样周期检测到滑移趋势，才判定为存在滑移风险）。
- 判定标准：滑移趋势检测准确率≥95%，无明显误判、漏判，滑移趋势出现后≤50ms内被检测到。

### 3.2.2 自动增稳控制
- 功能描述：当检测到物体存在滑移趋势时，算法自动启动增稳策略，通过调节夹持力、手指姿态，抑制滑移，确保抓取稳定。
- 具体要求：
  1. 增稳策略：优先通过微调关节角度来实现夹持力增稳（夹持力增加幅度为当前夹持力的10%-20%，且不超过物体安全夹持力阈值）；
  2. 增稳响应：滑移趋势检测确认后，≤50ms内启动增稳策略，增稳过程平稳，无明显冲击，避免因突然增力导致物体损伤。
- 判定标准：增稳策略启动后，≤100ms内抑制滑移趋势，物体无明显位移，后续保持稳定抓取，无二次滑移。

### 3.2.3 法向力超限卸力控制
- 功能描述：实时监测法向夹持力，当夹持力超过预设安全阈值（根据物体材质、重量设定）时，自动启动卸力策略，降低夹持力至安全范围，避免损伤物体或损坏灵巧手硬件。
- 具体要求：
  1. 力阈值设置：支持按物体类型预设法向力安全阈值，可手动配置、修改；默认设置紧急卸力阈值（超过安全阈值的120%时，启动紧急卸力）。
  2. 卸力控制：
    - 常规卸力：当法向力超过安全阈值但未达到紧急卸力阈值时，缓慢降低夹持力，加持力降低速度可控制，直至夹持力回落至安全阈值的80%-90%，保持稳定。
    - 紧急卸力：当法向力超过紧急卸力阈值时，快速卸力，直至夹持力回落至安全阈值，避免瞬间过载。
  3. 卸力反馈：卸力过程中，实时反馈法向力数据，卸力完成后，发出反馈信号（用于后续算法调整），若卸力后仍出现力超限，重复卸力并触发报警（可选）。
- 判定标准：法向力超限时，≤50ms内启动卸力策略，卸力后夹持力稳定在安全范围内，物体无损伤、无滑移。

## 3.3 辅助功能需求

### 3.3.1 异常处理
- 功能描述：针对抓取过程中的异常情况（如传感器故障、物体掉落、力超限无法卸力），具备异常处理能力，避免设备损坏或任务失败。
- 具体要求：
  1. 传感器故障：若力传感器、关节编码器数据异常（如数据突变、无数据），算法自动停止抓取动作，触发报警信号，记录故障信息。
  2. 物体掉落：若检测到物体掉落（夹持力突然降至0，且位移突变），自动松开手指，停止控制，手指张开，触发报警信号。
  3. 力超限无法卸力：若法向力超过紧急阈值，且卸力后仍无法回落至安全范围，自动切断灵巧手动力，停止夹持，触发报警信号。

### 3.3.3 参数可配置
- 功能描述：支持算法关键参数的手动配置、修改，适配不同场景、不同物体的抓取需求，无需修改算法代码。
- 可配置参数包括：安全系数、滑移检测阈值、法向力安全阈值、卸力速度等。
