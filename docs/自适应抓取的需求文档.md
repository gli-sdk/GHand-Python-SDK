# 自适应抓取功能需求文档

**日期**: 2026-04-21  
**功能**: 基于触觉反馈的自适应抓取  


## 1. 背景
具身机器人本体在使用灵巧手时，会用灵巧手取抓取物品，在搬运或移动的过程中，物品有滑脱的风险，而通过触觉传感器，能感知物体的滑移趋势，通过反馈微调关节角度，增加法向力，减少滑脱的情况。


## 2. 总体功能要求：
灵巧手运控算法稳定抓取模块需具备“感知-判断-控制-反馈”闭环能力，集成力感知、滑移检测、力调节三大核心功能，无需人工干预，自动适配不同材质、不同形状的抓取物体（预设常见物体参数库，支持参数扩展），确保抓取过程稳定、安全、可控。
## 2.1 前置条件
值得指出的是，实现该功能需要设置如下前提：
- 1）对于被抓取物体，其抓取姿态一般由“大脑”生成，在本功能中，抓取姿态为已知条件。
- 2）泛化性问题：一般情况下，需要针对不同的物体进行抓取进而验证自适应抓取的泛化性。在本功能中，将不同的物体抽象为几类进行泛化性的测试，计划采用圆柱、球体、立方体、圆环环面、四棱锥、香蕉、豆腐、金属铁块进行自适应抓取测试。

## 3. 详细功能需求：
### 3.1 稳定抓持功能
#### 3.1.1 抓取力自适应调节
- 功能描述：算法根据抓取物体的材质、重量、形状（预设参数或实时识别），自动计算并输出最优初始抓取力，避免抓取力过小导致物体滑移，或抓取力过大损伤物体。
- 具体要求：  
  1. 预设物体参数库：包含常见物体（如金属块、塑料件、水果）的材质硬度、重量、安全抓取力范围，支持手动添加、编辑物体参数。
  2. 初始抓取力设置：根据物体重量、材质硬度，自动计算初始抓取力或预设值抓取力，确保初始抓取力处于物体安全范围内。
  3. 抓取力校准：抓取过程中，实时采集手指力传感器数据，与预设安全范围对比，若初始抓取力偏差超过±5N，自动微调抓取力至合理范围。
- 判定标准：初始抓取力落在物体预设安全范围内，抓取后10s内物体无滑移、无明显变形（易损物体无破损）。

#### 3.1.2 抓取姿态稳定控制
- 功能描述：抓取过程中，算法实时调节灵巧手各关节角度，确保物体姿态保持预设角度（如水平、垂直），避免物体倾斜、翻转，实现姿态稳定。
- 具体要求：
  1. 人眼观测：稳定抓取后，人眼观察抓取物品的倾斜角度、翻转角度。
  2. 视觉传感器（可选）：实时采集物体姿态数据（倾斜角度、翻转角度）
- 判定标准：抓取后，人眼观察无明显倾斜、翻转或视觉传感器测量物体姿态倾斜角度≤5°、反转角度≤5°，持续保持稳定，无明显晃动。

#### 3.1.3 损伤防护控制
- 功能描述：针对易损物体（如豆腐、香蕉），算法具备损伤防护机制，避免抓取力过大导致物体破损、变形。
- 具体要求：
  1. 易损物体识别：支持通过参数库标记易损物体，或通过视觉传感器（可选）识别易损物体特征，自动切换至“损伤防护模式”。
  2. 抓取力限制：易损物体的最大抓取力不得超过预设安全阈值（可配置，根据物体材质设定），一旦接近阈值（达到阈值的90%），通过相关措施（减少关节角度、减小关节角度增量）减少抓取力/抓取力的增量，避免物体变形、变形。
  3. 关节动作限制：损伤防护模式下，灵巧手关节动作速度降低30%，避免动作过快产生冲击，损伤物体。
- 判定标准：易损物体抓取后无破损、无变形，抓取力始终≤预设安全阈值。

## 3.2 动态增稳功能
### 3.2.1 滑移趋势检测
- 功能描述：抓取过程中（包括静态夹持、动态移动过程），实时检测物体是否存在滑移趋势，及时触发增稳策略，避免物体滑移掉落。
- 具体要求：
  1. 滑移检测参数：通过手指力传感器采集摩擦力数据（触觉传感器的合力和分力），综合判断滑移趋势
  2. 检测频率：≥60Hz，确保滑移趋势及时被捕捉，避免延迟导致物体滑移。
  3. 误判防护：排除因手指微小振动、传感器噪声导致的误检测，设置防抖阈值（如连续3个采样周期检测到滑移趋势，才判定为存在滑移风险）。
- 判定标准：滑移趋势检测准确率≥95%，无明显误判、漏判，滑移趋势出现后≤50ms内被检测到。

### 3.2.2 自动增稳控制
- 功能描述：当检测到物体存在滑移趋势时，算法自动启动增稳策略，通过调节灵巧手关节角度，抑制滑移，确保抓取稳定。
- 具体要求：
  1. 增稳策略：优先通过微调关节角度来实现抓取力增稳；
  2. 增稳响应：滑移趋势检测确认后，≤50ms内启动增稳策略，增稳过程平稳，无明显冲击，避免因突然增力导致物体损伤。
- 判定标准：增稳策略启动后，≤0.5s内抑制滑移趋势，物体无明显位移，后续保持稳定抓取，无二次滑移。

### 3.2.3 时间超限释放保护
- 功能描述：在自适应保持阶段，系统持续监测抓持时长。当抓持时间达到预设上限时，无论当前触觉状态是否稳定、是否存在滑移趋势，均自动触发安全释放流程，驱动灵巧手张开至安全姿态，防止电机长时间运行温度累积过大，烧毁电机。（补充：后期若硬件支持，将通过读取电机温度传感器，以电机温度为释放保护信号，进行释放保护）
- 具体要求：
  1. 超时阈值设定：支持通过配置参数设定最大保持时长，默认值为 20秒，可根据实际要求进行设定。该阈值在自适应保持阶段开始计时时生效，一旦进入该状态即启动计时器。
  2. 超时检测与触发：检测频率与控制周期同步，每个周期检查一次已持续时间。当时间超时时，立即触发超时释放。时间超限释放的优先级高于滑移增稳和法向力超限卸力——即超时发生后不再执行任何收紧或微调动作，直接进入释放流程。
  3. 安全释放执行：触发后，采用位置模式下发张开姿态。
 
- 判定标准：达到超时阈值后，≤ 50 ms内启动释放流程；释放完成后，灵巧手所有主动关节回到张开姿态，电机电流回落至空载水平.

## 3.3 辅助功能需求

### 3.3.1 异常处理
- 功能描述：针对抓取过程中的异常情况（如传感器故障、物体掉落、灵巧手空抓），具备异常处理能力，避免设备损坏或任务失败。
- 具体要求：
  1. 传感器故障：若触觉力传感器无连接、灵巧手关节角度数据异常（如数据突变、无数据），算法自动停止抓取动作，触发报警信号，记录故障信息。
  
  2. 灵巧手空抓：当灵巧手闭合找接触时，灵巧手关节动作角度大于阈值且触觉传感器没有接触信号，自动松开手指，停止控制，手指张开，触发报警信号。
  
  3. 抓取物体丢失：物体掉落时/人为移走抓取物品，触觉传感器从有接触信号跳变至无接触信号，自动停止抓取动作，手指张开，触发报警信号，记录故障信息。

### 3.3.3 参数可配置
- 功能描述：支持算法关键参数的手动配置、修改，适配不同场景、不同物体的抓取需求，无需修改算法代码。
- 可配置参数包括：安全系数、滑移检测阈值、法向力安全阈值等。

## 4.量化指标
| 指标项 | 目标值 | 说明 |
|:---|:---|:---|
| 单手指抓取法向力量程 | 0.5 N ~ 15 N | 覆盖从易损柔性件到刚度大固体的抓取力范围 |
| 触觉反馈更新周期 | < 0.2 s | 触觉传感器刷新率 83.3 Hz|
| 接触时间 | < 1 s | 从预抓取姿态到检测到有效接触 |
|控制周期|≤50ms|自适应保持阶段的控制周期|
